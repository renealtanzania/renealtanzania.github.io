Executive Summary

Reneal, a United States NGO, has been partnering since 2013 with Arusha District to install computer labs in government secondary schools. In 2020, Reneal’s work extended to Longido District and Arusha City Council. In 2021 the expansion done in Handeni District, Tanga and 2022 Kondoa District in Dodoma. Reneal US relies heavily on Reneal Team Tanzania to execute its programs, a key reason for success to date.

User Features

  • Many applications (office suite, photo/video editing, webpage design, desktop publishing, CAD, programming, etc.).
  • Many educational programs (math, science, English, geography).
  • Extensive educational content (books, movies, offline Wikipedia, past National Exam papers, Tanzania curriculum notes, science simulations, etc.) stored on the server, shared by web browser to each client.
  • Epoptes classroom management program so teacher can monitor student work or share their screen to students (very convenient if school does not have a projector), shared video streaming capability.
  • TeachersShared and AllUsersShared file storage areas for permission-controlled content sharing.

    • System Design

  • Uses Linux and Open Source Software plus freely available educational content.
  • Client-server architecture, with all programs running on the server.
  • Usually start with 17 seats, with capacity to expand up to about 40 seats.
  • Network cabling connects clients through a network switch to the server.
  • Internet connection, if available at the school, plugs directly into the server.
  • Custom software for server configuration, accounts management, and internet control (filtering and blocking) if internet is available.
  • Reneal’s entire system can be shared with a USB hard drive and used to easily create new servers.
  • Three levels of user accounts for security and ease-of-access to personal files - System administration, Teacher and Student accounts.
